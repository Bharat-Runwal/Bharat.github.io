Hand-written digit Recognition

Automated handwritten digit recognition is widely used today - from recognizing zip codes (postal codes) on mail envelopes to recognizing amounts written on bank checks.A neural network is implemented to recognize handwritten digits using the given training set .The neural network will be able to represent complex models that form non-linear hypotheses.

The backpropagation algorithm is used to learn the parameters for the neural network.

The feedforward propagation and the parameters learned from backpropagation algorithm are used to predict handwritten digits.

With lambda=1 and Iterations = 400 Network acheives 99.52% accuracy

Spam-Classification

Many email services today provide spam filters that are able to classify emails into spam and non-spam email with high accuracy. This project will use SVMs to build spam filter.

Firstly all the emails are preprocessed according to given examples and then suitable normalization steps are taken like:- Lower-casing,Stripping HTML,Normalizing URLs,Word Stemming etc.

A Vocabulary list is selected by choosing all words which occur at least a 100 times in the spam corpus, resulting in a list of words.This list help in extracting features from Email.

The classifier gets a training accuracy of about 99.8% and a test accuracy of about 98.5%.

Steganography

Image Steganography

Steganography is a technique of hiding secret data within an ordinary non-secret file or message to avoid any detection, the secret data is then extracted at its destination.It can be used to conceal almost any type of digital content including text,image ,video or audio content etc.Here i have done Image steganography i.e. embedding secret message in an image.

There are several ways in which this can be done i have used Least significant bit method.Also,to increase the security of message a bit encryption method is also used.
